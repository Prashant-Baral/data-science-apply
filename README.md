# Data Files

## Source
- Original dataset: [London Bike Sharing Dataset](https://cycling.data.tfl.gov.uk)

## Files
- `df_main.csv` - Main cleaned dataset (89.85 MB)
- `df_anomaly.csv` - Anomaly detection results
- `df_invlaid.csv` - Invalid/filtered records

## Files in this repository
- `df_main_sample.csv` - Sample of main dataset (first 1000 rows)
- `df_anomaly_sample.csv` - Sample of anomaly detection results
- `df_invlaid_sample.csv` - Sample of invalid/filtered records

## Full Dataset
The full datasets are not included due to size constraints (df_main.csv is 89.85 MB).

To generate the full datasets:
1. Download the original London Bike Sharing dataset
2. Run `01_EDA_cleaning.ipynb`

The samples provided here are sufficient to understand the data structure and test the notebooks.
